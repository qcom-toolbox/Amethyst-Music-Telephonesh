# Amethyst Music - Release 1.0

Amethyst Music reaches its first major milestone with **Version 1.0**. This release solidifies the core experience, providing a stable and feature-rich music streaming client for Android, built with the latest modern technologies.

---

## 📱 Technical Specifications

| Requirement | Details |
|-------------|---------|
| **App Version** | 1.0 (Code 1) |
| **Minimum Android Version** | Android 6.0 (API Level 23) |
| **Target Android Version** | Android 15 (API Level 36) |
| **User Interface** | Jetpack Compose (Material 3) |
| **Architecture** | MVVM with Kotlin Coroutines & Flow |

---

## ✨ Key Features

- 🎵 **Premium Playback**: Powered by **Android Media3 (ExoPlayer)** for seamless, high-quality audio streaming.
- 📺 **Google Cast**: Stream your music to TVs, speakers, and other Google Cast-enabled devices.
- 🎤 **Synchronized Lyrics**: Enjoy real-time, time-synced lyrics with automatic caching for offline viewing.
- 🎚️ **Pro Equalizer**: A built-in 5-band equalizer with Bass Boost to tailor your sound.
- 📥 **Offline Downloads**: Save tracks directly to your device and listen without an internet connection.
- 🎨 **Dynamic Theming**: Support for **Material You** (Android 12+) dynamic colors, plus presets like AMOLED, Midnight, Forest, and Crimson.
- 🛡️ **Advanced Connectivity**: Support for custom server URLs and an option to bypass HTTPS errors for self-signed certificates.
- 🛠️ **Admin Management**: Authorized users can upload, edit, and delete tracks directly from the app.
- 🔍 **Smart Discovery**: Quickly search and navigate your entire music library.

## 🛠️ Tech Stack

- **Media Handling**: `androidx.media3` (ExoPlayer, MediaSession, Cast)
- **Networking**: `OkHttp`
- **Image Loading**: `Coil` (Compose support)
- **Dependency Management**: Gradle Version Catalog
- **Asynchronous Work**: Kotlin Coroutines & Flow

## 🚀 Getting Started

1. **Connect**: Set your server URL (Purple Music or Amethyst Music backend).
2. **Login**: Authenticate with your credentials.
3. **Play**: Browse your library and start streaming.
